üìù 2. Elemen Teks & Container

A. Paragraf <p> (Block Element)

Fungsi: Khusus untuk menampilkan paragraf teks. Browser otomatis kasih spasi di atas & bawah setiap paragraf.

<p>Ini paragraf pertama dengan spasi otomatis.</p>
<p>Ini paragraf kedua, browser otomatis kasih jarak.</p>

Ini paragraf kedua, browser otomatis kasih jarak.

B. Span <span> (Inline Element)

Fungsi: Untuk styling teks kecil/pendek di dalam kalimat tanpa membuat baris baru.

Saya suka apel dan jeruk.

C. Div <div> (Block Container)

Fungsi: Wadah besar untuk mengelompokkan elemen lain, digunakan untuk membuat layout seperti sidebar, header, atau section.

  <h3>Judul Konten</h3>
  <p>Div adalah wadah untuk mengelompokkan elemen.</p>
</div>

üñºÔ∏è 14. Frameset (Membagi Halaman)

‚ö†Ô∏è PERINGATAN: Frameset adalah teknologi LAMA dan sudah DEPRECATED (tidak direkomendasikan) di HTML5. Browser modern masih support tapi tidak disarankan untuk project baru. Gunakan <iframe> atau CSS Grid/Flexbox untuk layout modern.

Apa itu Frameset?

Frameset digunakan untuk membagi browser window menjadi beberapa bagian (frame), di mana setiap frame bisa load halaman HTML terpisah.

Struktur Dasar Frameset

<!DOCTYPE html>
<html>
<head>
  <title>Frameset Example</title>
</head>
<!-- TIDAK ADA <body> di frameset! -->
<frameset cols="25%, 75%">
  <frame src="menu.html" name="menu">
  <frame src="content.html" name="content">
</frameset>
</html>

Atribut Frameset:

Atribut Nilai Penjelasan
cols "25%, 50%, 25%"
"100, *, 200"		 Membagi frame secara VERTIKAL (kolom kiri-kanan)
rows "100, *, 50"
"20%, 60%, 20%"		 Membagi frame secara HORIZONTAL (baris atas-bawah)
border "0" atau "5" Ketebalan border antar frame (pixel)
frameborder "0" atau "1" Tampilkan/sembunyikan border frame
framespacing "10" Jarak antar frame (pixel)
bordercolor "#FF0000" Warna border frame

Atribut Frame:

Atribut Fungsi Nilai
src URL halaman yang di-load src="page.html"
name Nama frame (untuk target link) name="content"
noresize Frame tidak bisa di-resize user noresize="noresize"
scrolling Kontrol scrollbar "yes", "no", "auto"
frameborder Border frame individual "0" atau "1"
marginwidth Margin kiri-kanan (pixel) marginwidth="10"
marginheight Margin atas-bawah (pixel) marginheight="10"
bordercolor Warna border frame individual bordercolor="#00FF00"

Contoh Frameset Horizontal (Rows)

<frameset rows="20%, 60%, 20%">
  <frame src="header.html" name="header" noresize scrolling="no">
  <frame src="main.html" name="main" scrolling="auto">
  <frame src="footer.html" name="footer" noresize scrolling="no">
</frameset>

Contoh Frameset Vertikal (Cols)

<frameset cols="200, *">
  <frame src="menu.html" name="menu" noresize>
  <frame src="content.html" name="content">
</frameset>

Frameset Nested (Bersarang)

<frameset rows="100, *">
  <frame src="header.html" name="header">
  <frameset cols="200, *">
    <frame src="menu.html" name="menu">
    <frame src="content.html" name="content">
  </frameset>
</frameset>

Noframes (Fallback)

Fungsi: Konten yang ditampilkan jika browser tidak support frameset.

<frameset cols="25%, 75%">
  <frame src="menu.html">
  <frame src="content.html">
  <noframes>
    <body>
      <p>Browser Anda tidak support frames.</p>
    </body>
  </noframes>
</frameset>

Link Target untuk Frame

Saat menggunakan frameset, link bisa ditargetkan ke frame tertentu:

<!-- Di menu.html -->
<a href="page1.html" target="content">Halaman 1</a>
<a href="page2.html" target="content">Halaman 2</a>
<a href="page3.html" target="_blank">Buka Tab Baru</a>
<a href="page4.html" target="_top">Keluar dari Frame</a>
Target Fungsi
target="nama_frame" Buka di frame dengan nama tertentu
target="_self" Buka di frame yang sama
target="_blank" Buka di tab/window baru
target="_parent" Buka di parent frameset
target="_top" Keluar dari semua frame, isi full window
Kenapa Frameset Deprecated?
‚Ä¢ Sulit untuk SEO (search engine tidak suka frames)
‚Ä¢ Masalah bookmark (URL tidak berubah)
‚Ä¢ Tidak responsive untuk mobile
‚Ä¢ Sulit untuk print halaman
‚Ä¢ Masalah aksesibilitas untuk screen reader

Alternatif Modern:
‚Ä¢ Gunakan <iframe> untuk embed konten
‚Ä¢ Gunakan CSS Grid atau Flexbox untuk layout
‚Ä¢ Gunakan JavaScript frameworks (React, Vue, dll) untuk SPA
